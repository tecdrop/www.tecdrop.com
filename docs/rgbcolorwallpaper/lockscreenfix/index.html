RGB Color Wallpaper » Lock screen fix

By default, the Android wallpaper that is displayed on the Home screens, is also displayed on the Lock screen. However, some Android device manufacturers have decided to use a separate image for the Lock screen.

(On Android 7.0 and above, RGB Color Wallpaper uses the new system functionality to set the color wallpaper on both the home screen and the lock screen.)

If after running RGB Color Wallpaper, your lock screen is still not filled with color, follow the following simple steps:

1. Download the color wallpaper


Don't worry, the image size should be only a few kilobytes.

2. Set the Lock screen wallpaper

Go to Android Settings » Display » Lock screen wallpaper, and pick the color-wallpaper.png image you have just downloaded.

That's it.

Is there an easier way to do this?

Unfortunately no. Having a separate image for the Lock screen is not the Android way of doing things, so there is no standard programmatic way to access this functionality from an app.
