Pitch Black Wallpaper » Lock screen not black?

By default, the Android wallpaper that is displayed on the Home screens, is also displayed on the Lock screen. However, some Android device manufacturers have decided to use a separate image for the Lock screen.

(On Android 7.0 and above, Pitch Black Wallpaper uses the new system functionality to set both the home screen and lock screen wallpaper to pitch black.)

If after running Pitch Black Wallpaper, your lock screen is still not black (pitch black), follow the following simple steps:

1. Download the Pitch Black image

If the image does not download when tapping the button, long tap it instead and select Save Link. Don't worry, the image size is less than half a kilobyte!

Download

2. Set the Lock screen wallpaper

Go to Android Settings » Display » Lock screen wallpaper, and pick the pitch-black-image.png image you have just downloaded.

That's it.

Is there an easier way to do this?

Unfortunately no. Having a separate image for the Lock screen is not the Android way of doing things, so there is no standard programmatic way to access this functionality from an app.
